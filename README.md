## Requisites

- **Node** (recent vesion)
- **pnpm** (package manager for Node)

## Set Up

This repo has a `Makefile` to facilitate the use of the most useful command for this blog.

1. Install libraries

```bash
make install
```

> To see all the make command list run with: `make help`
